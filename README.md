# Number Transformation Search Agent

Ένας αυτόνομος πράκτορας αναζήτησης σε Python που βρίσκει τη βέλτιστη ακολουθία πράξεων για να μετασχηματίσει έναν αρχικό αριθμό σε έναν αριθμό-στόχο.

## Υποστηριζόμενοι Αλγόριθμοι
Το πρόγραμμα υλοποιεί και συγκρίνει τις εξής μεθόδους αναζήτησης:
* **Breadth-First Search (BFS)**: Για εύρεση της λύσης με τα λιγότερα βήματα.
* **Depth-First Search (DFS)**: Εξερεύνηση σε βάθος.
* **Best-First Search**: Αναζήτηση με βάση την ευρετική συνάρτηση (heuristic).
* **A* Search**: Συνδυασμός κόστους διαδρομής και ευρετικής για τη βέλτιστη λύση.

## Διαθέσιμες Πράξεις
Ο πράκτορας μπορεί να εκτελέσει τις εξής πράξεις σε κάθε βήμα:
* **Increase (+1)** | **Decrease (-1)**
* **Double (x2)** | **Half (//2)**
* **Square (x²)** | **Root (√x)**

Κάθε πράξη έχει ένα δυναμικό κόστος που υπολογίζεται βάσει της τιμής του αριθμού.

## Τεχνολογίες
* **Python 3.x**
* **Heaps (heapq)**: Για την αποτελεσματική διαχείριση της ουράς προτεραιότητας στον A*.

## Οδηγίες Χρήσης
Η εκτέλεση γίνεται από το τερματικό με την εξής δομή:
`python register.py <method> <start_val> <target_val> <output_file>`

**Παράδειγμα:**
