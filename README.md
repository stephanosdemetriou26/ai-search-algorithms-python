# Number Transformation Search Agent

Ένας αυτόνομος πράκτορας αναζήτησης σε Python που βρίσκει τη βέλτιστη ακολουθία πράξεων για να μετασχηματίσει έναν αρχικό αριθμό σε έναν αριθμό-στόχο.

## Υποστηριζόμενοι Αλγόριθμοι
Το πρόγραμμα υλοποιεί και συγκρίνει τις εξής μεθόδους αναζήτησης:
* **Breadth-First Search (BFS)**: Για εύρεση της λύσης με τα λιγότερα βήματα[cite: 7].
* **Depth-First Search (DFS)**: Εξερεύνηση σε βάθος[cite: 7].
* **Best-First Search**: Αναζήτηση με βάση την ευρετική συνάρτηση (heuristic)[cite: 7].
* **A* Search**: Συνδυασμός κόστους διαδρομής και ευρετικής για τη βέλτιστη λύση[cite: 7].

## Διαθέσιμες Πράξεις
Ο πράκτορας μπορεί να εκτελέσει τις εξής πράξεις σε κάθε βήμα:
* **Increase (+1)** | **Decrease (-1)**[cite: 7]
* **Double (x2)** | **Half (//2)**[cite: 7]
* **Square (x²)** | **Root (√x)**[cite: 7]

Κάθε πράξη έχει ένα δυναμικό κόστος που υπολογίζεται βάσει της τιμής του αριθμού[cite: 7].

## Τεχνολογίες
* **Python 3.x**[cite: 7]
* **Heaps (heapq)**: Για την αποτελεσματική διαχείριση της ουράς προτεραιότητας στον A*[cite: 7].

## Οδηγίες Χρήσης
Η εκτέλεση γίνεται από το τερματικό με την εξής δομή:
`python register.py <method> <start_val> <target_val> <output_file>`

**Παράδειγμα:**
`python register.py astar 5 18 solution.txt`[cite: 7]
